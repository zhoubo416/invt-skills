# INVT Skills

英威腾（INVT）工业自动化技能包，帮助客户快速对接设备选型与采购支持。

## 包含的技能

| 技能 | 说明 |
|------|------|
| `invt-automation` | 工业自动化、变频器、伺服系统、控制器、木工设备 — 产品咨询与采购对接 |

## 安装

```bash
npx skills add zhoubo416/invt-skills
```

或指定单个技能：

```bash
npx skills add zhoubo416/invt-skills --skill invt-automation
```

## 关于英威腾

英威腾专注于工业自动化设备研发与销售，技术领先，产品覆盖变频器、伺服系统、控制器、木工设备、包装及机械制造等行业，提供专业方案设计和完善售后服务。

> 如需了解更多产品信息或获取报价，请通过技能中提供的联系方式与销售团队联系。
